# Titanic Dataset Analysis

## 📌 Overview
This repository contains analysis and exploration of the famous **Titanic dataset**, originally made available through [Kaggle](https://www.kaggle.com/c/titanic). The dataset provides information about passengers aboard the Titanic, including demographics, ticket details, and survival outcomes. It is widely used for practicing data analysis, machine learning, and interview preparation.

---

## 📂 Dataset Description
The dataset consists of two main CSV files:

- **train.csv**: Includes passenger details and survival labels.
- **test.csv**: Includes passenger details without survival labels (used for predictions).

### Key Columns:
- `PassengerId`: Unique identifier for each passenger
- `Survived`: Survival indicator (0 = No, 1 = Yes)
- `Pclass`: Ticket class (1 = Upper, 2 = Middle, 3 = Lower)
- `Name`: Passenger’s full name
- `Sex`: Gender
- `Age`: Age in years
- `SibSp`: Number of siblings/spouses aboard
- `Parch`: Number of parents/children aboard
- `Ticket`: Ticket number
- `Fare`: Ticket fare
- `Cabin`: Cabin number
- `Embarked`: Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton)

## 🎯 Objectives
- Perform **exploratory data analysis (EDA)** to understand passenger demographics and survival patterns.
- Apply **data cleaning** techniques (handling missing values, encoding categorical variables).
- Build **machine learning models** to predict survival outcomes.
- Document findings and insights for interview preparation and portfolio demonstration.
